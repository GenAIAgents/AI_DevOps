## Setup app
Create virtual environment:  
`python -m venv .venv`

Activate created venv:  
- Windows  
`.venv\Scripts\activate.bat`

- macOS and Linux  
`source .venv/bin/activate`

Setup python dependencies:  
`pip install -r requirements.txt`


## Run app
- Windows  
`python -m streamlit run src\ui\app.py`

- macOS and Linux  
`python -m streamlit run src/ui/app.py`
